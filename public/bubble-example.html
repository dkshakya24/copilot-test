How to Use in SharePoint

Just add this single line to your SharePoint Script Editor:

<script src="https://copilot-test-theta.vercel.app/copilot-bubble.js"></script>

That's it! The bubble will automatically appear.

Optional: Customization

You can customize the bubble before loading the script:

<script> window.CopilotBubbleConfig = { position: 'bottom-right', primaryColor: '#2563eb', bubbleText: 'AI', apiEndpoint: 'https://your-api.com' }; </script> <script src="https://copilot-test-theta.vercel.app/copilot-bubble.js"></script>

Programmatic Control

You can control the bubble programmatically:

// Open the chat CopilotBubble.open(); // Close the chat CopilotBubble.close(); // Toggle the chat CopilotBubble.toggle(); // Show notification badge CopilotBubble.showBadge(3); // Check if open CopilotBubble.isOpen();
