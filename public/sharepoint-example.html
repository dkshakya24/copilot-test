SharePoint Integration Example

Copy and paste this code into your SharePoint Script Editor Web Part

Method 1: Using Widget Script (Recommended)

This is the easiest method. Simply copy the code below into a SharePoint Script Editor Web Part:

<!-- Replace 'your-domain.com' with your actual domain --> <div id="copilot-container"></div> <script src="https://your-domain.com/copilot-widget.js"></script> <script> window.CopilotConfig = { containerId: 'copilot-container', iframeSrc: 'https://your-domain.com/embed', position: 'bottom-right', theme: 'auto', apiEndpoint: 'https://your-api.com', height: '600px', width: '400px', autoLoad: true }; </script>
Note: Replace your-domain.com with your actual production domain. Make sure your Next.js app is deployed and accessible via HTTPS.

Method 2: Direct Iframe Embedding

For simple embedding without the widget script:

<div style="position: fixed; bottom: 20px; right: 20px; z-index: 9999; width: 400px; height: 600px;"> <iframe src="https://your-domain.com/embed?theme=auto&position=bubble" style="width: 100%; height: 100%; border: none; border-radius: 12px; box-shadow: 0 4px 20px rgba(0,0,0,0.15);" title="AI Copilot Chat" allow="microphone; camera"> </iframe> </div>
